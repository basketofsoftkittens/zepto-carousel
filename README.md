# Zepto and jQuery Mobile Carousel

http://jcemer.com/zepto-carousel

Very simple carousel with support to *swipe* and *css transitions*.

``` html
<div class="container">
    <ul>
        <li>Page 1</li>
        <li>Page 2</li>
        <li>Page 3</li>
        <li>Page 4</li>
    </ul>
</div>
```

``` javascript
$('.container ul').carousel();
```
